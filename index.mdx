examplenv is a CLI tool that scans your project for `.env` files and produces safe, committable `example.env` counterparts. It preserves comments, whitespace, and non-secret values — and redacts secrets using simple inline directives.

## How it works

examplenv reads every `.env*` file in your project, processes each line according to any redaction directives you've added, and writes a matching `example.env*` file alongside it.

<Steps>
  <Step title="Add directives to your .env files">
    Mark secret variables with `# !SECRET` or wrap sensitive regions with `# !MASK-ON` / `# !MASK-OFF`.
  </Step>
  <Step title="Run examplenv gen">
    Run `examplenv gen` from your project root. examplenv finds all `.env*` files recursively and generates corresponding `example.env*` files.
  </Step>
  <Step title="Commit the example files">
    The generated files contain placeholder values like `<YOUR_OPENAI_API_KEY>` for secrets, and real values for everything else — safe to commit to version control.
  </Step>
</Steps>
